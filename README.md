# fullstack-demo-project

Instalar dependências de front e back separadamente.

1 - Fazer o clone
2 - Abrir terminal
3 - cd back
4 - Instalar dependências
5 - nest start
6 - Abrir segundo terminal
7 - cd front
8 - instalar dependências
9 - npm run dev
