# fullstack-demo-project

Instalar dependências de front e back separadamente.

1 - Fazer o clone  <br/>
2 - Abrir terminal  <br/>
3 - cd back  <br/>
4 - Instalar dependências (yarn add ou pnpm install)  <br/>
5 - nest start  <br/>
6 - Abrir segundo terminal  <br/>
7 - cd front  <br/>
8 - instalar dependências (yarn add ou pnpm install)  <br/>
9 - npm run dev  <br/>

http://localhost:4444/docs  <br/>
